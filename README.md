# 🏋️‍♂️ Fitness Tracker App

An intuitive Android application designed to help users log their body metrics, visualize weight trends, and stay on top of their fitness goals. With a clean, modern UI, users can easily onboard, set target weights, and track their daily progress.

## ✨ Features

1. **Secure Authentication**: Seamless login and sign-up flows with support for both standard Email/Password and Google Sign-In integration.
2. **Personalized Onboarding**: A step-by-step setup process ("Body Data") that collects baseline metrics like gender and height (with support for both FT and CM).
3. **Dynamic Dashboard**: A central hub that provides an at-a-glance view of the user's current Weight, Height, and automatically calculated BMI.
4. **Interactive Weight Trends**: A visual line chart that tracks historical weight logs against a user-defined goal weight, making it easy to see progress over time.
5. **Daily Logging**: A quick and accessible action button to log the day's current weight.

## 🛠️ Tech Stack (Suggested)

* **UI Toolkit**: Jetpack Compose
* **Language**: Kotlin
* **Architecture**: MVVM (Model-View-ViewModel)
* **Backend/Auth**: Firebase Authentication / Ktor
* **Local Storage**: Room Database / DataStore (for caching logs and preferences)
